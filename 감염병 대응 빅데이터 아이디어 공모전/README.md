## <a id='data_analysis'><b>데이터 분석 방법론</b></a>
해당 아이디어 공모전의 경우 , 감염병이라는 큰 틀 안에서 주제를 정하여 데이터 분석을 진행해야 하기 때문에</b>
현재 문제가 되고 있는 현황 , 도메인 지식등을 미리 조사하면서 먼저 주제를 정하고 , 데이터를 모은 후에 데이터를 이해하는 과정이 필요했기에 , 방법론으로는
CRISP-DM 방식으로 채택을 함.
